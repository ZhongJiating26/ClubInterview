## 🧹 推荐清理操作

### 方案 A：安全清理（推荐）

**只删除 Python 缓存文件：**
```powershell
# 在后端目录运行
cd E:\Projects\ClubInterview\backend
# 删除 Python 缓存
for /d /r . in (__pycache__) do @rmdir /s /q "."
```

---
